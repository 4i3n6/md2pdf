MD2PDF Manual

MD2PDF is a free Markdown to PDF converter with no limits and no signup required. Write in Markdown in the editor, preview in real-time in A4 format, and export directly to PDF.

In this manual you'll find everything you need to master the tool: from getting started to advanced export features and offline use.

Quick Tips

#
Headings with #

Use # for headings. More # = smaller level (## subtitle, ### section).

Ctrl
Ctrl+Shift+E

Quick shortcut to export your document to PDF instantly.

[PB]
Page break

Use to force a page break in the PDF. Click [PB] button to insert.

MER
Mermaid Diagrams

Create flowcharts and diagrams using ```mermaid blocks. Click MER button to insert template.

PWA
Install the app

MD2PDF works offline. Install as an app on your device.
