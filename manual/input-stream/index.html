Área de Edição (INPUT_STREAM)

O Editor de Markdown

O painel INPUT_STREAM é onde você escreve e edita seu conteúdo Markdown. O editor oferece recursos avançados para facilitar a escrita:

  • Syntax highlighting em tempo real: O código Markdown é colorido automaticamente, facilitando a identificação de títulos, negrito, itálico, links e outros elementos.
  • Números de linha para referência: Cada linha do editor é numerada, facilitando a navegação e referência em documentos longos.
  • Indicadores de erro: Linhas onduladas aparecem sob elementos com problemas de sintaxe, ajudando você a identificar e corrigir erros rapidamente.

Gerenciando Documentos

A sidebar DOC_INDEX à esquerda mostra todos os seus documentos salvos. O gerenciamento é simples e intuitivo:

  • DOC_INDEX na sidebar: Lista todos os documentos disponíveis no navegador.
  • Clique + para criar novo documento: O botão de adição cria um documento em branco pronto para edição.
  • Clique no nome para abrir: Selecione qualquer documento da lista para carregá-lo no editor.
  • Clique [x] para deletar: Remove o documento permanentemente. Uma confirmação é solicitada antes da exclusão.
  • Documentos salvos automaticamente: Todas as alterações são salvas automaticamente no armazenamento local do navegador.

Renomeando Documentos

Personalizar o nome do documento é importante, pois ele será usado como nome do arquivo PDF exportado:

  • Clique no campo de nome: O campo de nome do documento está localizado acima do editor.
  • Digite o novo nome: Edite o texto diretamente no campo.
  • Nome salvo automaticamente: Não é necessário confirmar; o nome é atualizado instantaneamente.
  • Nome usado como nome do arquivo PDF: Ao exportar, o PDF terá o mesmo nome do documento.

Botão COPY

O botão COPY permite copiar rapidamente todo o conteúdo Markdown do documento atual:

  • Localização: Ao lado do campo de nome do documento.
  • Função: Copia todo o conteúdo Markdown para a área de transferência.
  • Atalho: Ctrl+Shift+C para copiar sem usar o mouse.
  • Uso: Ideal para compartilhar o texto, fazer backup ou continuar a edição em outro editor.

Indicadores de Erro

O editor identifica problemas de sintaxe Markdown e os sinaliza visualmente:

  • Linha ondulada vermelha: Indica um erro de sintaxe que precisa ser corrigido.
  • Linha ondulada amarela: Indica um aviso sobre algo que pode estar incorreto.
  • Passe o mouse para ver mensagem: Ao posicionar o cursor sobre o indicador, uma mensagem explica o problema.
