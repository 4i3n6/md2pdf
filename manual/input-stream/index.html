Área de Edição (INPUT_STREAM)

O Editor de Markdown

O painel INPUT_STREAM é onde você escreve e edita seu conteúdo Markdown.

  • Syntax highlighting: Código Markdown colorido automaticamente
  • Números de linha: Facilita referência em documentos longos
  • Indicadores de erro: Linhas onduladas mostram problemas de sintaxe

Gerenciando Documentos

A sidebar DOC_INDEX mostra todos os seus documentos salvos.

  • Clique + para criar: Novo documento em branco
  • Clique no nome: Abre o documento
  • Clique [x]: Deleta (com confirmação)
  • Salvamento automático: Alterações salvas instantaneamente

Renomeando Documentos

  • Clique no campo de nome acima do editor
  • Digite o novo nome
  • O nome é usado como nome do PDF exportado

Botão COPY

  • Localização: Ao lado do nome do documento
  • Atalho: Ctrl+Shift+C
  • Função: Copia todo o Markdown para a área de transferência
