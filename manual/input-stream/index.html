INPUT_STREAM - Área de Edição

Editor de Markdown com syntax highlighting e gerenciamento de documentos

O Editor de Markdown

O editor de markdown do MD2PDF oferece uma experiência de escrita profissional com recursos avançados para facilitar a criação de documentos.

  • Real-time Syntax Highlighting - O código markdown é colorido automaticamente enquanto você digita, facilitando a identificação de headers, listas, links, código e outros elementos.
  • Line Numbers - Números de linha são exibidos na lateral esquerda do editor, facilitando a navegação e referência a trechos específicos do documento.
  • Error Indicators - Linhas onduladas (wavy underlines) aparecem automaticamente para indicar problemas de sintaxe no seu markdown.

Gerenciando Documentos

O painel lateral DOC_INDEX permite gerenciar todos os seus documentos de forma simples e intuitiva.

  • Visualizar documentos - O sidebar DOC_INDEX exibe a lista de todos os documentos salvos no navegador.
  • Criar novo documento - Clique no botão + para criar um novo documento em branco.
  • Abrir documento - Clique no nome do documento na lista para abri-lo no editor.
  • Excluir documento - Clique no botão [x] ao lado do documento para excluir. Uma confirmação será solicitada antes da exclusão.
  • Auto-save - Os documentos são salvos automaticamente no armazenamento do navegador (localStorage). Não é necessário salvar manualmente.

Renomeando Documentos

O nome do documento pode ser alterado a qualquer momento diretamente na interface.

  • Editar nome - Clique no campo de nome do documento no topo do editor.
  • Digitar novo nome - Digite o novo nome desejado para o documento.
  • Salvamento automático - O nome é salvo automaticamente ao sair do campo ou pressionar Enter.
  • Nome como arquivo PDF - O nome do documento será utilizado como nome do arquivo PDF ao exportar.

Botão COPY

O botão COPY permite copiar rapidamente todo o conteúdo markdown do documento atual.

  • Localização - O botão está posicionado ao lado do campo de nome do documento, na barra superior do editor.
  • Função - Copia todo o conteúdo markdown do documento para a área de transferência.
  • Atalho de teclado - Use Ctrl + Shift + C para copiar rapidamente.
  • Casos de uso - Compartilhar markdown com outras pessoas, fazer backup do conteúdo, ou editar em outro aplicativo.

Indicadores de Erro

O editor identifica automaticamente problemas na sintaxe markdown e os destaca visualmente.

Indicador Significado Exemplo
Linha vermelha ondulada Erro de sintaxe que pode afetar a renderização texto com erro
Linha amarela ondulada Aviso sobre possível problema ou sugestão de melhoria texto com aviso

Dica: Passe o mouse sobre a linha ondulada para ver a mensagem detalhada do erro ou aviso.

Dicas
  • Use nomes significativos - Nomeie seus documentos de forma descritiva para facilitar a organização e identificação posterior.
  • Erros não bloqueiam exportação - Mesmo com erros indicados no editor, você ainda pode exportar o documento para PDF. Os erros são apenas indicativos.
