Offline Use and PWA

What is PWA?

  • Progressive Web App
  • Website that works like native app
  • Installable on device
  • Works 100% offline

How to Install

Desktop (Chrome/Edge)

  • Visit /app
  • Click install icon in address bar
  • Confirm "Install"

Android

  • Menu (3 dots) → "Add to home screen"

iOS

  • Share → "Add to Home Screen"

Offline Operation

  • After first access, app is cached
  • All features available offline
  • Saved documents remain accessible
  • Automatic updates when online
