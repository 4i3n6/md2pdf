Exportar para PDF

Botão EXP_PDF

O botão [ EXP_PDF ] é a forma principal de exportar seu documento para PDF.

  • Localização: Canto superior direito da interface, na barra de controles.
  • Atalho: Ctrl+Shift+E para acesso rápido sem usar o mouse.
  • Ação: Abre o diálogo de impressão nativo do navegador.

Passo a Passo

Siga estas etapas para exportar seu documento:

  1. Verifique se o preview está como desejado Confira o painel RENDER_OUTPUT para garantir que a formatação está correta.
  2. Clique em [ EXP_PDF ] ou use Ctrl+Shift+E O diálogo de impressão do navegador será aberto automaticamente.
  3. Selecione "Salvar como PDF" Em "Destino" ou "Impressora", escolha a opção para salvar como arquivo PDF.
  4. Nome do arquivo O sistema usa automaticamente o nome do seu documento como sugestão.
  5. Clique em Salvar Escolha o local de destino e confirme para gerar o PDF final.

Configurações Recomendadas

Para obter os melhores resultados na exportação:

  • Destino: Salvar como PDF (não selecione uma impressora física).
  • Margens: Padrão ou Mínimas, dependendo da sua preferência.
  • Escala: 100% para manter o tamanho original.
  • Gráficos de fundo: Ative se quiser preservar cores de fundo.

Download MD

Além do PDF, você pode baixar o arquivo Markdown original:

  • Localização: Botão na sidebar, na seção ACTIONS.
  • Formato: Baixa o arquivo em formato .md (Markdown puro).
  • Utilidade: Ideal para backup ou edição em outros editores.

Validação Pré-impressão

O MD2PDF realiza verificações automáticas antes da exportação:

  • Verificação automática: Analisa o documento em busca de problemas.
  • Avisos no SYS_LOGS: Problemas detectados aparecem no painel de logs.
  • Exportação opcional: Você pode continuar mesmo com avisos.

Solução de Problemas

PDF diferente do preview?

Verifique se o zoom do navegador está em 100%.

Conteúdo faltando no PDF?

Role o preview completamente para verificar se todo o conteúdo está sendo renderizado.

Páginas em branco?

Verifique se há separadores --- extras no conteúdo.
