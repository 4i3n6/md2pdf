Export to PDF

How to Export

  1. Check the preview Verify formatting is correct in the right panel.
  2. Click EXP_PDF or Ctrl+Shift+E Opens the browser's print dialog.
  3. Select "Save as PDF" In Destination/Printer, choose save as PDF.
  4. Click Save Document name is used as file name.

Recommended Settings

  • Destination: Save as PDF
  • Margins: Default or Minimum
  • Scale: 100%
  • Background graphics: Enable if you want colors
