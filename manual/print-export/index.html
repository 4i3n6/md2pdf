Exportar para PDF

Botão EXP_PDF

O botão [ EXP_PDF ] está localizado no canto superior direito da interface.

Você também pode usar o atalho de teclado Ctrl+Shift+E para acionar a exportação rapidamente.

Ao clicar no botão ou usar o atalho, o diálogo de impressão do navegador será aberto automaticamente.

Passo a Passo

  1. Verifique o preview

    Certifique-se de que a visualização A4 está exatamente como desejado antes de exportar.

  2. Clique em [ EXP_PDF ]

    Ou use o atalho Ctrl+Shift+E para abrir o diálogo de impressão.

  3. Selecione "Salvar como PDF"

    No diálogo de impressão, escolha a opção "Salvar como PDF" como destino.

  4. Nome do arquivo

    O nome sugerido para o arquivo será automaticamente o nome do seu documento.

  5. Clique em Salvar

    Escolha o local desejado e confirme para gerar o PDF.

Configurações Recomendadas

No diálogo de impressão do navegador, recomendamos as seguintes configurações:

Configuração Valor Recomendado
Destino Salvar como PDF
Margens Padrão ou Mínimas
Escala 100%
Gráficos de fundo Ativado (se quiser preservar cores de fundo)

Download MD

Além de exportar para PDF, você pode baixar o arquivo Markdown original.

O botão de download está localizado na barra lateral, na seção ACTIONS.

Essa função é útil para:

  • Fazer backup do seu documento
  • Editar em outro editor de sua preferência
  • Compartilhar o código-fonte Markdown

Validação Pré-impressão

Antes de exportar, o sistema verifica automaticamente seu documento em busca de potenciais problemas.

Caso algum problema seja detectado, avisos serão exibidos no SYS_LOGS (logs do sistema).

Problemas Comuns Detectados

  • Imagens muito grandes - podem aumentar significativamente o tamanho do PDF
  • URLs muito longas - podem quebrar o layout ou ficar cortadas
  • Conteúdo excessivo - documentos muito extensos podem demorar para processar

Nota: Os avisos são apenas informativos. Você pode prosseguir com a exportação mesmo que existam avisos.

Solução de Problemas

Problema Solução
PDF está diferente do preview Verifique se o zoom do navegador está em 100%
Conteúdo faltando Role o preview para verificar se todo o conteúdo está visível
Páginas em branco Verifique se há separadores --- extras no conteúdo
