Exportar para PDF

Como Exportar

  1. Verifique o preview Confira se a formatação está correta no painel direito.
  2. Clique EXP_PDF ou Ctrl+Shift+E Abre o diálogo de impressão do navegador.
  3. Selecione "Salvar como PDF" Em Destino/Impressora, escolha salvar como PDF.
  4. Clique Salvar O nome do documento é usado como nome do arquivo.

Configurações Recomendadas

  • Destino: Salvar como PDF
  • Margens: Padrão ou Mínimas
  • Escala: 100%
  • Gráficos de fundo: Ativar se quiser cores

Download MD

Baixe o arquivo Markdown original na seção ACTIONS da sidebar. Ideal para backup.

Problemas Comuns

PDF diferente do preview?

Verifique se o zoom do navegador está em 100%.

Páginas em branco?

Verifique se há --- extras no conteúdo.
