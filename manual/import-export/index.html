Import and Export Files

Importing Markdown Files

Bring your existing .md files into MD2PDF.

Supported formats

.md, .markdown, and .txt files are accepted.

  1. Click IMPORT MD In the sidebar, find the ACTIONS section.
  2. Select your file Choose a .md, .markdown, or .txt file from your computer.
  3. Edit and preview Your content is loaded into a new document.

Exporting as Markdown

Save your document as a .md file for backup or use elsewhere.

  1. Click EXPORT MD In the sidebar ACTIONS section.
  2. Save the file Browser downloads the .md file with your document name.

Backup and Restore

Create a full backup of all documents and preferences.

Backup format

JSON file with documents, preferences, and metadata.

  1. Click BACKUP ALL In the sidebar ACTIONS section.
  2. Save the file Keep the .json file in a safe place.
  3. Restore when needed Click RESTORE, select the backup file, and confirm. This replaces current documents.

Exporting as PDF

For PDF export, see the dedicated Export PDF page.

Copying Content

Use the COPY button or Ctrl+Shift+C to copy all Markdown to clipboard. Perfect for pasting into other apps.
