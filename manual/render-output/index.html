RENDER_OUTPUT - Preview A4

A área de preview do MD2PDF exibe seu documento Markdown renderizado em formato A4, exatamente como ele aparecerá quando exportado para PDF. Acompanhe suas alterações em tempo real enquanto digita.

Preview em Tempo Real

O preview do MD2PDF atualiza automaticamente conforme você digita no editor. Não é necessário salvar ou clicar em nenhum botão para ver as mudanças.

  • Atualização instantânea: As alterações aparecem imediatamente enquanto você digita
  • Indicador verde: O ponto verde no cabeçalho mostra o status de renderização
  • Métrica LATENCY: Exibe o tempo de renderização em milissegundos na barra superior

Formato A4

O preview simula uma folha de papel A4, o formato padrão para impressão e documentos.

  • Dimensões: O preview simula papel A4 (210x297mm)
  • Margens: As margens já estão pré-configuradas para impressão
  • WYSIWYG: O que você vê é muito próximo do que será impresso

Como Imagens são Tratadas

O MD2PDF processa imagens automaticamente para garantir que elas se ajustem corretamente ao formato A4.

  • Redimensionamento automático: Imagens são automaticamente redimensionadas para caber no A4
  • Imagens grandes: Imagens que excedem a largura da página são reduzidas proporcionalmente
  • Quebras inteligentes: Imagens evitam quebras de página quando possível

Quebras de Página

Controle onde as páginas do seu PDF serão divididas usando quebras manuais ou deixe o sistema gerenciar automaticamente.

  • Quebra manual: Use --- (linha horizontal) para forçar uma quebra de página
  • Blocos de código: Blocos de código e imagens tentam permanecer em uma única página
  • Listas longas: Listas extensas podem ser divididas entre páginas automaticamente

Elementos Renderizados

O preview suporta todos os elementos padrão do Markdown, além de algumas extensões:

  • Títulos: Hierarquia de títulos (h1 a h6) com estilos diferenciados
  • Texto formatado: Negrito, itálico e tachado
  • Listas: Listas com marcadores e listas numeradas
  • Blocos de código: Com syntax highlighting para diversas linguagens
  • Tabelas: Tabelas com bordas e estilização
  • Citações: Blockquotes com estilo visual diferenciado
  • Links: Links clicáveis que funcionam no PDF final

Dicas

!
Aproximação visual

O preview não é 100% exato - o PDF final pode variar ligeiramente devido a diferenças de renderização do navegador.
