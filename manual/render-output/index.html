RENDER_OUTPUT - Preview A4

O preview mostra seu documento em formato A4, exatamente como aparecerá no PDF.

Preview em Tempo Real

  • Atualização instantânea: Muda enquanto você digita
  • Indicador verde: Status de renderização no cabeçalho
  • Métrica LATENCY: Tempo de renderização em ms

Formato A4

  • Dimensões: Simula papel A4 (210x297mm)
  • Margens: Pré-configuradas para impressão
  • WYSIWYG: Muito próximo do resultado final

Imagens

  • Redimensionamento automático para caber no A4
  • Imagens grandes são reduzidas proporcionalmente
  • Evitam quebras de página quando possível

Quebras de Página

  • Manual: Use --- para forçar quebra
  • Automática: Blocos de código tentam ficar inteiros
  • Listas longas: Podem ser divididas automaticamente
