RENDER_OUTPUT - A4 Preview

The preview shows your document in A4 format, exactly as it will appear in the PDF.

Real-time Preview

  • Instant update: Changes as you type
  • Green indicator: Render status in header
  • LATENCY metric: Render time in ms

A4 Format

  • Dimensions: Simulates A4 paper (210x297mm)
  • Margins: Pre-configured for printing
  • WYSIWYG: Very close to final result

Page Breaks

  • Manual: Use <!-- pagebreak --> or click [PB] button
  • Visual indicator: Dashed box shows where page will break
  • Automatic: Code blocks try to stay together
