Privacidade e Armazenamento

Seus documentos ficam armazenados APENAS no seu navegador. O MD2PDF não tem acesso aos seus dados. Nada é enviado para servidores.

100% Local

  • Todo processamento acontece no seu navegador
  • Nenhum servidor recebe seus dados
  • Zero rastreamento ou analytics
  • Sem contas, sem login

Onde Seus Dados Ficam

  • Armazenados no localStorage do navegador
  • Chave: md2pdf-docs-v2
  • Dados ficam apenas no seu dispositivo
  • Navegadores diferentes = armazenamento diferente

Limite de Armazenamento

  • localStorage: ~5MB por domínio
  • Monitore uso: métrica MEM na barra superior
  • Imagens grandes aumentam uso
  • Se atingir limite, delete documentos antigos

Como Limpar Seus Dados

  • Opção 1: Delete documentos individualmente
  • Opção 2: Limpe dados do navegador para este site
  • Opção 3: DevTools > Application > localStorage > delete

Backup de Documentos

  • Use DOWNLOAD MD para salvar arquivos
  • Use botão COPY para copiar conteúdo
  • Faça backups regulares de docs importantes

Sigilo Garantido

  • Sem cookies de rastreamento
  • Sem scripts de terceiros
  • Funciona 100% offline

Atenção: Se você limpar dados do navegador, os documentos serão PERDIDOS. Navegadores diferentes não compartilham dados. Modo privado/anônimo pode não salvar dados.
