Privacidade e Armazenamento

100% Local. Seus documentos ficam APENAS no seu navegador. Nada é enviado para servidores.

Privacidade Total

  • Processamento 100% no navegador
  • Zero rastreamento ou analytics
  • Sem contas, sem login
  • Funciona offline

Armazenamento

  • Dados no localStorage do navegador
  • Limite: ~5MB por domínio
  • Métrica MEM mostra uso atual
  • Navegadores diferentes = dados separados

Backup

  • Use DOWNLOAD MD para salvar arquivos
  • Use COPY para copiar conteúdo
  • Faça backups regulares!

Atenção: Limpar dados do navegador APAGA seus documentos. Modo privado/anônimo pode não salvar.
