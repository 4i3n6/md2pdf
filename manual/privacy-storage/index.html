Privacidade e Armazenamento Local

Seus documentos ficam armazenados APENAS no seu navegador.

O MD2PDF não tem acesso aos seus dados.

Nada é enviado para servidores.

100% Local

  • Todo o processamento acontece no seu navegador
  • Nenhum processamento em servidor
  • Nenhuma transmissão de dados
  • Sem contas, sem rastreamento, sem analytics

Onde Seus Dados Ficam

  • Armazenados no localStorage do navegador
  • Chave de armazenamento: md2pdf-docs-v2
  • Os dados permanecem apenas no seu dispositivo
  • Navegadores diferentes = armazenamento diferente

Limite de Armazenamento

  • Limite do localStorage: aproximadamente 5MB por domínio
  • Monitore o uso: métrica MEM na barra superior
  • Imagens grandes aumentam o uso de armazenamento
  • Se o limite for atingido, delete documentos antigos

Como Limpar Seus Dados

1
Delete documentos individualmente

Use o botão de exclusão na lista de documentos

2
Limpe os dados do navegador para este site

Nas configurações do navegador, limpe dados do site md2pdf

3
Use o DevTools

Abra DevTools > Application > localStorage > delete

Backup de Documentos

  • Use DOWNLOAD MD para salvar arquivos markdown
  • Use o botão COPY para copiar o conteúdo
  • Considere fazer backups regulares de documentos importantes

Sigilo Garantido

  • Código fonte é aberto e auditável
  • Sem cookies para rastreamento
  • Sem scripts de terceiros
  • Funciona completamente offline

Aviso Importante

  • Se você limpar os dados do navegador, os documentos serão PERDIDOS
  • Navegadores diferentes não compartilham dados entre si
  • O modo Privado/Anônimo pode não salvar os dados
