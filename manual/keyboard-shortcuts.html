Atalhos de Teclado

Use atalhos de teclado para trabalhar mais rapido. Os atalhos abaixo permitem criar documentos, exportar PDFs, navegar entre documentos e editar texto sem tirar as maos do teclado.

Atalhos da Aplicacao

Atalhos globais do MD2PDF para gerenciamento de documentos e exportacao.

Atalho Acao
Ctrl + N Criar novo documento
Ctrl + Shift + E Exportar PDF (abre dialogo de impressao)
Ctrl + Shift + C Copiar conteudo Markdown
Ctrl + Shift + P Preview de impressao (toggle)
↑ / ↓ Navegar entre documentos na lista
Enter Abrir documento selecionado
Delete Deletar documento selecionado
Escape Sair do modo preview de impressao

Atalhos do Editor

Atalhos padrao do CodeMirror para edicao de texto.

Atalho Acao
Ctrl + Z Desfazer
Ctrl + Shift + Z Refazer
Ctrl + A Selecionar tudo
Ctrl + F Buscar (se disponivel)

Usuarios Mac

No Mac, substitua Ctrl por Cmd (⌘).

Por exemplo: Cmd + N para criar novo documento, Cmd + Shift + E para exportar PDF.
