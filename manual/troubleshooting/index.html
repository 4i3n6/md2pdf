Troubleshooting Common Issues

PDF Issues

PDF looks different from preview?

Make sure browser zoom is at 100%. Check print settings: margins, scale, background graphics.

Extra blank pages?

Check for extra --- in your content. Each creates a page break.

Images not showing in PDF?

Use direct image URLs (https://). Local file paths won't work in PDF.

Formatting Issues

Markdown not rendering?

Check for spaces: # Title needs space after #. **bold** needs no spaces inside.

Links not working?

Use full URLs with protocol: https://example.com

Code not highlighted?

Add language after backticks: ```javascript

Storage Issues

Documents disappeared?

Did you clear browser data? Use a different browser? Documents are per-browser.

Can't save (MEM full)?

Export old documents and delete them to free space.

Private/Incognito mode?

Documents may not persist. Use regular mode.

Offline Issues

App not working offline?

Visit the app online first to cache it. Then offline will work.

Can't install as PWA?

Only works on HTTPS. Use Chrome, Edge, or Safari.

App not updating?

Clear site data and reload, or wait for auto-update.

Still having issues?

Check that you're using a modern browser (Chrome, Edge, Firefox, Safari). Clear cache and reload the page. Most issues are resolved by a fresh start.
