Exportar para PDF

Botao EXP_PDF

O botao [ EXP_PDF ] esta localizado no canto superior direito da interface.

Voce tambem pode usar o atalho de teclado Ctrl+Shift+E para acionar a exportacao rapidamente.

Ao clicar no botao ou usar o atalho, o dialogo de impressao do navegador sera aberto automaticamente.

Passo a Passo

  1. Verifique o preview

    Certifique-se de que a visualizacao A4 esta exatamente como desejado antes de exportar.

  2. Clique em [ EXP_PDF ]

    Ou use o atalho Ctrl+Shift+E para abrir o dialogo de impressao.

  3. Selecione "Salvar como PDF"

    No dialogo de impressao, escolha a opcao "Salvar como PDF" como destino.

  4. Nome do arquivo

    O nome sugerido para o arquivo sera automaticamente o nome do seu documento.

  5. Clique em Salvar

    Escolha o local desejado e confirme para gerar o PDF.

Configuracoes Recomendadas

No dialogo de impressao do navegador, recomendamos as seguintes configuracoes:

Configuracao Valor Recomendado
Destino Salvar como PDF
Margens Padrao ou Minimas
Escala 100%
Graficos de fundo Ativado (se quiser preservar cores de fundo)

Download MD

Alem de exportar para PDF, voce pode baixar o arquivo Markdown original.

O botao de download esta localizado na barra lateral, na secao ACTIONS.

Essa funcao e util para:

  • Fazer backup do seu documento
  • Editar em outro editor de sua preferencia
  • Compartilhar o codigo-fonte Markdown

Validacao Pre-impressao

Antes de exportar, o sistema verifica automaticamente seu documento em busca de potenciais problemas.

Caso algum problema seja detectado, avisos serao exibidos no SYS_LOGS (logs do sistema).

Problemas Comuns Detectados

  • Imagens muito grandes - podem aumentar significativamente o tamanho do PDF
  • URLs muito longas - podem quebrar o layout ou ficar cortadas
  • Conteudo excessivo - documentos muito extensos podem demorar para processar

Nota: Os avisos sao apenas informativos. Voce pode prosseguir com a exportacao mesmo que existam avisos.

Solucao de Problemas

Problema Solucao
PDF esta diferente do preview Verifique se o zoom do navegador esta em 100%
Conteudo faltando Role o preview para verificar se todo o conteudo esta visivel
Paginas em branco Verifique se ha separadores --- extras no conteudo
