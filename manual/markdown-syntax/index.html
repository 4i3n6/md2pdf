Guia Completo de Markdown

Nesta página

Esta página é sua referência completa de Markdown. Consulte sempre que precisar lembrar como formatar algo.

Títulos

Use # para criar títulos. Quanto mais #, menor o nível do título.

# Título Principal (H1)
## Subtítulo (H2)
### Seção (H3)
#### Subseção (H4)
##### Menor (H5)
###### Mínimo (H6)

Os títulos H1 e H2 são os mais usados. H3 e H4 servem para subdivisões.

Dica

Sempre deixe um espaço entre o # e o texto. #Título não funciona, use # Título.

Formatação de Texto

Formate palavras ou frases com símbolos ao redor do texto.

Atenção

Não deixe espaço entre os símbolos e o texto. ** texto ** não funciona.

Você escreve Resultado Uso
**texto**textoDestaque importante
*texto*textoÊnfase sutil
***texto***textoMáximo destaque
~~texto~~textoTexto removido/errado
`texto`textoCódigo ou comando
==texto==textoMarcação (highlight)

Quebras de Linha e Parágrafos

Markdown trata quebras de linha de forma especial. Entender isso evita frustrações.

Esta linha termina com dois espaços  
e continua aqui na linha de baixo.

Este é um novo parágrafo
porque tem uma linha em branco acima.

---

Isso força uma nova página no PDF.

Os dois espaços no final são invisíveis mas necessários para quebra de linha simples.

O que você quer Como fazer
Nova linha (mesmo parágrafo)Dois espaços no final da linha + Enter
Nova linha (alternativa)<br> no final da linha
Novo parágrafoLinha em branco entre os textos
Quebra de página (PDF)--- em linha separada
Importante

Se você apertar Enter uma vez sem os dois espaços, o Markdown junta as linhas no mesmo parágrafo.

Listas

Crie listas com marcadores ou números.

Lista com marcadores

- Item um
- Item dois
  - Subitem (2 espaços antes)
  - Outro subitem
- Item três

Use -, * ou + para marcadores.

Lista numerada

1. Primeiro item
2. Segundo item
   1. Subitem numerado
   2. Outro subitem
3. Terceiro item

A numeração é automática. Você pode usar 1. em todos os itens.

Lista de tarefas

- [ ] Tarefa pendente
- [x] Tarefa concluída
- [ ] Outra pendente

Use [ ] para pendente e [x] para concluído.

Links e Imagens

Adicione links clicáveis e imagens ao seu documento.

Imagens no MD2PDF

Imagens são redimensionadas automaticamente para caber no A4. Use URLs completas ou cole imagens em base64.

Tipo Sintaxe Exemplo
Link simples[texto](url)[Google](https://google.com)
Link com título[texto](url "título")[Google](https://google.com "Buscador")
URL direta<url><https://google.com>
Imagem![alt](url)![Logo](imagem.png)
Imagem com link[![alt](img)](url)Imagem clicável

Código

Mostre código com formatação especial e syntax highlighting.

Linguagens suportadas

javascript, typescript, python, java, c, cpp, csharp, php, ruby, go, rust, sql, html, css, json, yaml, bash, markdown e muitas outras.

Código inline

Use `console.log()` para debug.

Envolva com crases simples para código no meio do texto.

Bloco de código

```javascript
function hello() {
  console.log("Hello!");
}
```

Use três crases e opcionalmente o nome da linguagem para syntax highlighting.

Citações (Blockquotes)

Use > para criar blocos de citação.

> Esta é uma citação.
> Pode ter múltiplas linhas.
>
> E parágrafos separados.

>> Citação aninhada (dois >)

Citações são úteis para destacar informações importantes ou citar fontes.

Tabelas

Crie tabelas usando | e -.

| Coluna 1 | Coluna 2 | Coluna 3 |
|----------|:--------:|---------:|
| Esquerda | Centro   | Direita  |
| Texto    | Texto    | Texto    |

Use : para alinhar: :--- esquerda, :---: centro, ---: direita.

Dica

As colunas não precisam estar alinhadas no código. O Markdown entende mesmo desalinhado.

Caracteres Especiais

Alguns caracteres têm significado especial no Markdown. Para usá-los literalmente, escape com \.

Caractere Escape Quando escapar
*\*Quando não quer itálico/negrito
#\#Quando não quer título
[ ]\[ \]Quando não quer link
`\`Quando não quer código
|\|Dentro de tabelas
\\\Para mostrar a barra

Dicas Avançadas

Truques úteis para documentos mais elaborados.

Copie e teste!

A melhor forma de aprender é experimentar. Copie os exemplos desta página, cole no MD2PDF e veja o resultado.

  • Linha horizontal: Use ---, *** ou ___ em linha separada. No MD2PDF, isso também força quebra de página.
  • Comentários: Use <!-- comentário --> para texto que não aparece no resultado.
  • HTML inline: Você pode usar tags HTML como <br>, <sub>, <sup>, <kbd>.
  • Emojis: Cole emojis diretamente ou use códigos como :smile: (suporte varia).
  • Notas de rodapé: Use [^1] no texto e [^1]: Nota aqui no final.
  • Âncoras: Títulos geram IDs automáticos. Link com [texto](#titulo-em-minusculas).
