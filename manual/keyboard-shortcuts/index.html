Atalhos de Teclado

Use atalhos para trabalhar mais rápido.

Atalhos da Aplicação

Atalho Ação
Ctrl+N Criar novo documento
Ctrl+Shift+E Exportar PDF
Ctrl+Shift+C Copiar conteúdo Markdown
Ctrl+Shift+P Preview de impressão (toggle)
↑ / ↓ Navegar entre documentos
Enter Abrir documento selecionado
Delete Deletar documento selecionado
Escape Sair do preview de impressão

Atalhos do Editor

Atalho Ação
Ctrl+Z Desfazer
Ctrl+Shift+Z Refazer
Ctrl+A Selecionar tudo

Nota para Usuários Mac

Substitua Ctrl por Cmd (⌘)
