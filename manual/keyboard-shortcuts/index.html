Keyboard Shortcuts

Use shortcuts to work faster.

Application Shortcuts

Shortcut Action
Ctrl+N Create new document
Ctrl+S Force save
Ctrl+Shift+E Export PDF
Ctrl+Shift+C Copy Markdown content
Ctrl+Shift+P Print preview
↑ / ↓ Navigate documents
Enter Open selected document
Delete Delete document
Escape Exit preview

Editor Shortcuts

Shortcut Action
Ctrl+Z Undo
Ctrl+Shift+Z Redo
Ctrl+A Select all

Mac Users

Replace Ctrl with Cmd (⌘)
