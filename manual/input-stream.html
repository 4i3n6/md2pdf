INPUT_STREAM - Area de Edicao

Editor de Markdown com syntax highlighting e gerenciamento de documentos

O Editor de Markdown

O editor de markdown do MD2PDF oferece uma experiencia de escrita profissional com recursos avancados para facilitar a criacao de documentos.

  • Real-time Syntax Highlighting - O codigo markdown e colorido automaticamente enquanto voce digita, facilitando a identificacao de headers, listas, links, codigo e outros elementos.
  • Line Numbers - Numeros de linha sao exibidos na lateral esquerda do editor, facilitando a navegacao e referencia a trechos especificos do documento.
  • Error Indicators - Linhas onduladas (wavy underlines) aparecem automaticamente para indicar problemas de sintaxe no seu markdown.

Gerenciando Documentos

O painel lateral DOC_INDEX permite gerenciar todos os seus documentos de forma simples e intuitiva.

  • Visualizar documentos - O sidebar DOC_INDEX exibe a lista de todos os documentos salvos no navegador.
  • Criar novo documento - Clique no botao + para criar um novo documento em branco.
  • Abrir documento - Clique no nome do documento na lista para abri-lo no editor.
  • Excluir documento - Clique no botao [x] ao lado do documento para excluir. Uma confirmacao sera solicitada antes da exclusao.
  • Auto-save - Os documentos sao salvos automaticamente no armazenamento do navegador (localStorage). Nao e necessario salvar manualmente.

Renomeando Documentos

O nome do documento pode ser alterado a qualquer momento diretamente na interface.

  • Editar nome - Clique no campo de nome do documento no topo do editor.
  • Digitar novo nome - Digite o novo nome desejado para o documento.
  • Salvamento automatico - O nome e salvo automaticamente ao sair do campo ou pressionar Enter.
  • Nome como arquivo PDF - O nome do documento sera utilizado como nome do arquivo PDF ao exportar.

Botao COPY

O botao COPY permite copiar rapidamente todo o conteudo markdown do documento atual.

  • Localizacao - O botao esta posicionado ao lado do campo de nome do documento, na barra superior do editor.
  • Funcao - Copia todo o conteudo markdown do documento para a area de transferencia.
  • Atalho de teclado - Use Ctrl + Shift + C para copiar rapidamente.
  • Casos de uso - Compartilhar markdown com outras pessoas, fazer backup do conteudo, ou editar em outro aplicativo.

Indicadores de Erro

O editor identifica automaticamente problemas na sintaxe markdown e os destaca visualmente.

Indicador Significado Exemplo
Linha vermelha ondulada Erro de sintaxe que pode afetar a renderizacao texto com erro
Linha amarela ondulada Aviso sobre possivel problema ou sugestao de melhoria texto com aviso

Dica: Passe o mouse sobre a linha ondulada para ver a mensagem detalhada do erro ou aviso.

Dicas
  • Use nomes significativos - Nomeie seus documentos de forma descritiva para facilitar a organizacao e identificacao posterior.
  • Erros nao bloqueiam exportacao - Mesmo com erros indicados no editor, voce ainda pode exportar o documento para PDF. Os erros sao apenas indicativos.
