O que é Markdown?

Entenda por que sua IA escreve com símbolos estranhos e como transformá-los em documentos bonitos.

Por que sua IA escreve assim?

Se você já usou ChatGPT, Claude, Gemini ou outras IAs, provavelmente notou que as respostas vêm com símbolos estranhos como **, #, -, ` e outros.

Isso não é um bug! Esses símbolos são formatação Markdown - uma linguagem simples que transforma texto puro em documentos formatados.

Markdown foi criado para ser legível mesmo sem renderização. Por isso as IAs usam esse formato: é universal, simples e funciona em qualquer lugar.

Markdown em 2 minutos

Veja como os símbolos se transformam em formatação:

Você escreve Resultado
# Título Título principal (grande)
## Subtítulo Subtítulo (médio)
**texto** Texto em negrito
*texto* Texto em itálico
- item Lista com marcadores
1. item Lista numerada
`código` Código inline
[link](url) Link clicável
> citação Bloco de citação
Dica: Você não precisa memorizar nada! Basta colar o texto no MD2PDF e ver a mágica acontecer.

Como usar o MD2PDF

Converter texto de IA para PDF é simples:

  1. Copie o texto da IA
    Selecione toda a resposta do ChatGPT, Claude, Gemini ou qualquer outra IA e copie (Ctrl+C).
  2. Cole na área INPUT_STREAM
    Abra o MD2PDF e cole o texto no editor à esquerda. O painel se chama "INPUT_STREAM (MD)".
  3. Veja o preview em tempo real
    O painel direito "RENDER_OUTPUT" mostra instantaneamente como seu documento ficará formatado.
  4. Exporte para PDF
    Clique no botão [ EXP_PDF ] no topo da página. Seu documento será gerado em formato A4, pronto para imprimir ou compartilhar.

Dicas para melhores resultados

  • Copie a resposta completa: Não copie apenas partes do texto. O Markdown funciona melhor quando o contexto completo está presente.
  • Use --- para quebras de página: Adicione três hifens em uma linha separada para criar uma nova página no PDF.
  • Revise o preview antes de exportar: O painel RENDER_OUTPUT mostra exatamente como o PDF ficará. Ajuste o texto se necessário.
  • Salve seus documentos: O MD2PDF salva automaticamente no navegador. Crie novos documentos clicando no botão + na sidebar.
