# Padrões de Escrita

## Português Brasileiro (PT-BR)

### REGRA CRÍTICA: Acentuação
- **SEMPRE** usar acentuação correta em português
- Escrever sem acentos é considerado erro grave
- Exemplos corretos: não, está, é, você, será, código, índice, página

### Erros comuns a evitar:
- "nao" → "não"
- "voce" → "você"
- "pagina" → "página"
- "codigo" → "código"
- "tambem" → "também"
- "esta" (verbo) → "está"
- "e" (verbo ser) → "é"
- "sera" → "será"
- "indice" → "índice"
- "numero" → "número"

### Copywriting
- Evitar linguagem técnica quando possível
- Focar no problema do usuário, não na solução técnica
- Usuário comum não sabe o que é "Markdown" ou "símbolos"
- Preferir linguagem acessível e direta
