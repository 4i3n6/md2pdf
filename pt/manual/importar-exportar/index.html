Importar e Exportar Arquivos

Importando Arquivos Markdown

Traga seus arquivos .md existentes para o MD2PDF.

Formatos suportados

Arquivos .md, .markdown e .txt são aceitos.

  1. Clique IMPORT MD Na sidebar, encontre a seção ACTIONS.
  2. Selecione seu arquivo Escolha um arquivo .md, .markdown ou .txt do seu computador.
  3. Edite e visualize Seu conteúdo é carregado em um novo documento.

Exportando como Markdown

Salve seu documento como arquivo .md para backup ou uso em outros lugares.

  1. Clique EXPORT MD Na seção ACTIONS da sidebar.
  2. Salve o arquivo O navegador baixa o arquivo .md com o nome do seu documento.

Exportando como PDF

Para exportar PDF, veja a página dedicada Exportar PDF.

Copiando Conteúdo

Use o botão COPY ou Ctrl+Shift+C para copiar todo o Markdown. Perfeito para colar em outros apps.
