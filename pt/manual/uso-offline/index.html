Uso Offline e PWA

O que é PWA?

  • Progressive Web App
  • Site que funciona como app nativo
  • Instalável no dispositivo
  • Funciona 100% offline

Como Instalar

Desktop (Chrome/Edge)

  • Acesse /pt/app
  • Clique no ícone de instalação na barra de endereço
  • Confirme "Instalar"

Android

  • Menu (3 pontos) → "Adicionar à tela inicial"

iOS

  • Compartilhar → "Adicionar à Tela de Início"

Funcionamento Offline

  • Após primeiro acesso, app fica em cache
  • Todas as funcionalidades disponíveis offline
  • Documentos salvos continuam acessíveis
  • Atualizações automáticas quando online
