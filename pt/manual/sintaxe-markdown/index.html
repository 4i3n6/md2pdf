Guia Completo de Markdown

Nesta página
Títulos Formatação de Texto Quebras de Linha Listas Links e Imagens Código Citações Tabelas Caracteres Especiais Dicas Avançadas

Esta página é sua referência completa de Markdown. Consulte sempre que precisar lembrar como formatar algo.

Títulos

Use # para criar títulos. Quanto mais #, menor o nível do título.

# Título Principal (H1)
## Subtítulo (H2)
### Seção (H3)
#### Subseção (H4)
##### Menor (H5)
###### Mínimo (H6)

Os títulos H1 e H2 são os mais usados. H3 e H4 servem para subdivisões.

Dica

Sempre deixe um espaço entre o # e o texto. #Título não funciona, use # Título.

Formatação de Texto

Formate palavras ou frases com símbolos ao redor do texto.

Você escreve Resultado Uso
**texto**textoDestaque importante
*texto*textoÊnfase sutil
***texto***textoMáximo destaque
~~texto~~textoTexto removido/errado
`texto`textoCódigo ou comando
==texto==textoMarcação (highlight)

Quebras de Linha e Parágrafos

Markdown trata quebras de linha de forma especial.

O que você quer Como fazer
Nova linha (mesmo parágrafo)Dois espaços no final + Enter
Nova linha (alternativa)<br> no final
Novo parágrafoLinha em branco entre os textos
Quebra de página (PDF)--- em linha separada

Listas

Crie listas com marcadores ou números.

Lista com marcadores

- Item um
- Item dois
  - Subitem (2 espaços antes)
- Item três

Use -, * ou + para marcadores.

Lista numerada

1. Primeiro item
2. Segundo item
3. Terceiro item

A numeração é automática.

Lista de tarefas

- [ ] Tarefa pendente
- [x] Tarefa concluída

Use [ ] para pendente e [x] para concluído.

Links e Imagens

Adicione links clicáveis e imagens ao seu documento.

Tipo Sintaxe Exemplo
Link[texto](url)[Google](https://google.com)
Imagem![alt](url)![Logo](imagem.png)

Código

Mostre código com formatação especial.

Código inline

Use `console.log()` para debug.

Envolva com crases simples.

Bloco de código

```javascript
function hello() {
  console.log("Hello!");
}
```

Use três crases e o nome da linguagem.

Citações (Blockquotes)

Use > para criar blocos de citação.

> Esta é uma citação.
> Pode ter múltiplas linhas.

Citações são úteis para destacar informações.

Tabelas

Crie tabelas usando | e -.

| Coluna 1 | Coluna 2 |
|----------|----------|
| Dado 1   | Dado 2   |

Use : para alinhar colunas.

Caracteres Especiais

Para usar caracteres especiais literalmente, escape com \.

Caractere Escape
*\*
#\#
`\`

Dicas Avançadas

  • Linha horizontal: Use --- para separadores.
  • Comentários: Use <!-- comentário -->.
  • HTML inline: Tags como <br>, <kbd> funcionam.
