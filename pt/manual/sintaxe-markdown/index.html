Guia Completo de Markdown

Nesta página
Títulos Formatação de Texto Quebras de Linha Listas Links e Imagens

Esta página é sua referência completa de Markdown. Consulte sempre que precisar lembrar como formatar algo.

Títulos

Use # para criar títulos. Mais símbolos # = nível menor.

# Título Principal (H1)
## Subtítulo (H2)
### Seção (H3)
#### Subseção (H4)
##### Menor (H5)
###### Mínimo (H6)

H1 e H2 são os mais comuns. H3 e H4 são para subdivisões.

Dica

Sempre deixe um espaço entre # e o texto. #Título não funciona, use # Título.

Formatação de Texto

Formate palavras ou frases com símbolos ao redor do texto.

Você escreve Resultado Uso
**texto**textoÊnfase importante
*texto*textoÊnfase sutil
***texto***textoÊnfase máxima
~~texto~~textoTexto deletado/errado
`texto`textoCódigo ou comando
==texto==textoDestaque

Quebras de Linha e Parágrafos

Markdown trata quebras de linha de forma especial. Entender isso evita frustrações.

O que você quer Como fazer
Nova linha (mesmo parágrafo)Dois espaços no final + Enter
Nova linha (alternativa)<br> no final
Novo parágrafoLinha em branco entre textos
Quebra de página (PDF)<!-- pagebreak --> ou botão [PB]

Listas

Crie listas com marcadores ou números.

Lista com marcadores

- Item um
- Item dois
  - Subitem (2 espaços antes)
- Item três

Use -, * ou + para marcadores.

Lista numerada

1. Primeiro item
2. Segundo item
3. Terceiro item

A numeração é automática.

Lista de tarefas

- [ ] Tarefa pendente
- [x] Tarefa concluída

Use [ ] para pendente, [x] para concluído.

Links e Imagens

Adicione links clicáveis e imagens ao seu documento.

Imagens no MD2PDF

Imagens são redimensionadas automaticamente para caber no A4. Use URLs completas ou cole imagens base64.

Tipo Sintaxe Exemplo
Link[texto](url)[Google](https://google.com)
Imagem![alt](url)![Logo](imagem.png)
