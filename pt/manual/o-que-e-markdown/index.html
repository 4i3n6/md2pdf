O que é Markdown?

Uma Linguagem de Formatação Simples

Markdown é uma linguagem de marcação leve criada por John Gruber em 2004. Ela permite adicionar formatação a texto simples usando símbolos simples.

Diferente de HTML ou documentos Word, arquivos Markdown são legíveis mesmo sem software especial. Isso o torna perfeito para documentação, notas e conteúdo que precisa ser portável.

Onde o Markdown é Usado

  • GitHub & GitLab: Arquivos README, issues, pull requests
  • Respostas de IA: ChatGPT, Claude, Gemini formatam suas saídas em Markdown
  • Documentação: Docs técnicos, wikis, bases de conhecimento
  • Anotações: Obsidian, Notion, Bear e muitos outros
  • Blogs: Jekyll, Hugo e muitos geradores de sites estáticos
  • Mensagens: Slack, Discord, Reddit suportam Markdown

Por que Usar Markdown?

  • Portável: Texto simples funciona em qualquer lugar
  • À prova de futuro: Sem formato proprietário para se preocupar
  • Rápido: Sem clicar em menus, apenas digite
  • Legível: Faz sentido mesmo sem renderização
  • Universal: Mesma sintaxe em todas as plataformas

Markdown vs HTML

Markdown é convertido para HTML para exibição. Você escreve # Título e vira <h1>Título</h1>.

Markdown HTML Resultado
# Título<h1>Título</h1>Título grande
**negrito**<strong>negrito</strong>Texto negrito
*itálico*<em>itálico</em>Texto itálico
[link](url)<a href="url">link</a>Link clicável
