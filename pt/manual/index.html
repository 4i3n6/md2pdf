Manual do MD2PDF

O MD2PDF é um conversor de Markdown para PDF gratuito, sem limites e sem necessidade de cadastro. Escreva em Markdown no editor, visualize em tempo real no formato A4 e exporte diretamente para PDF.

Neste manual você encontrará tudo o que precisa para dominar a ferramenta: desde os primeiros passos até recursos avançados de exportação e uso offline.

Dicas Rápidas

#
Títulos com #

Use # para títulos. Mais # = nível menor (## subtítulo, ### seção).

Ctrl
Ctrl+Shift+E

Atalho rápido para exportar seu documento para PDF instantaneamente.

[PB]
Quebra de página

Use para forçar uma quebra de página no PDF. Clique no botão [PB] para inserir.

PWA
Instale o app

MD2PDF funciona offline. Instale como app no seu dispositivo.

MER
Diagramas Mermaid

Crie fluxogramas e diagramas usando blocos ```mermaid. Clique no botao MER para inserir template.
