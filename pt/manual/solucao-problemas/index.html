Solução de Problemas Comuns

Problemas com PDF

PDF diferente do preview?

Confirme que o zoom do navegador está em 100%. Verifique configurações de impressão: margens, escala, gráficos de fundo.

Páginas em branco extras?

Verifique se há --- extras no conteúdo. Cada um cria uma quebra de página.

Imagens não aparecem no PDF?

Use URLs diretas de imagens (https://). Caminhos de arquivos locais não funcionam no PDF.

Problemas de Formatação

Markdown não renderiza?

Verifique espaços: # Título precisa de espaço após #. **negrito** não pode ter espaços dentro.

Links não funcionam?

Use URLs completas com protocolo: https://exemplo.com

Código sem cores?

Adicione a linguagem após as crases: ```javascript

Problemas de Armazenamento

Documentos sumiram?

Limpou dados do navegador? Usou navegador diferente? Documentos são por navegador.

Não consegue salvar (MEM cheio)?

Exporte documentos antigos e delete-os para liberar espaço.

Modo privado/anônimo?

Documentos podem não persistir. Use modo normal.

Problemas Offline

App não funciona offline?

Visite o app online primeiro para fazer cache. Depois offline funcionará.

Não consegue instalar como PWA?

Só funciona em HTTPS. Use Chrome, Edge ou Safari.

App não atualiza?

Limpe dados do site e recarregue, ou aguarde atualização automática.

Ainda com problemas?

Confirme que está usando um navegador moderno (Chrome, Edge, Firefox, Safari). Limpe o cache e recarregue a página. A maioria dos problemas é resolvida com um novo começo.
