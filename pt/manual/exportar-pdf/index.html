Exportar para PDF

Como Exportar

  1. Verifique o preview Confirme que a formatação está correta no painel direito.
  2. Clique EXP_PDF ou Ctrl+Shift+E Abre o diálogo de impressão do navegador.
  3. Selecione "Salvar como PDF" Em Destino/Impressora, escolha salvar como PDF.
  4. Clique Salvar O nome do documento é usado como nome do arquivo.

Configurações Recomendadas

  • Destino: Salvar como PDF
  • Margens: Padrão ou Mínimas
  • Escala: 100%
  • Gráficos de fundo: Ativar se quiser cores
