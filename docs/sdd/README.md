# SDD - Indice

Este diretorio contem documentos SDD (Software Design Document) por funcionalidade.
Cada SDD segue o mesmo modelo para facilitar revisao e manutencao.

## Modelo SDD
1. Objetivo
2. Escopo
3. Componentes e Responsabilidades
4. Fluxo principal
5. Dados e Persistencia
6. Interfaces (UI/APIs)
7. Erros e Logs
8. Seguranca e Privacidade
9. Performance e Limites
10. Testes e Validacao
11. Riscos e Pendencias
